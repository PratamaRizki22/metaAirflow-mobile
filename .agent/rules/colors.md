## Design Rules

1. Always use these colors for consistency
2. No custom colors outside this palette
3. Use semantic colors for status/feedback
4. Maintain WCAG AA/AAA contrast ratios
5. Dark mode uses lighter/desaturated versions
6. Background should never be pure black (#000000)
7. Use Teal (#0D9488) as primary brand color
8. Use Mint Green (#6EE7B7) as accent/secondary

## Typography

- No custom fonts without approval

## Icons
- No custom icons (use emoji or text only)
